Prerequisites

Before you begin, ensure you have:

  • Python 3.9+ installed
  • pip package manager
  • Basic familiarity with async/await Python

Installation

Install Agent OS with pip:

# Basic installation
pip install agent-os

# With all optional dependencies
pip install agent-os[full]

# Specific modules only
pip install agent-os[cmvk]      # Cross-Model Verification
pip install agent-os[iatp]      # Inter-Agent Trust Protocol
pip install agent-os[healthcare] # Healthcare compliance

Verify the installation:

python -c "import agent_os; print(agent_os.__version__)"
# Output: 1.1.0

Your First Governed Agent

Let's create a simple agent with kernel-level governance:

from agent_os import KernelSpace

# 1. Create a kernel with the default strict policy
kernel = KernelSpace(policy="strict")

# 2. Register your agent function
@kernel.register
async def my_agent(task: str) -> str:
    """A simple agent that processes tasks."""
    # Your agent logic here
    # The kernel intercepts all actions and enforces policies
    return f"Completed: {task}"

# 3. Execute the agent safely
import asyncio

async def main():
    result = await kernel.execute(my_agent, "analyze this data")
    print(result)
    # Output: Completed: analyze this data

asyncio.run(main())

What happens behind the scenes:

  1. The kernel wraps your function with policy enforcement
  2. Every action is checked against the active policy
  3. Blocked actions are stopped before execution
  4. All actions are logged to the flight recorder

Using Policies

Policies define what agents can and cannot do. The strict policy blocks dangerous operations:

Built-in Policies

# Strict - blocks file writes, shell commands, etc.
kernel = KernelSpace(policy="strict")

# Permissive - allows most actions, logs everything
kernel = KernelSpace(policy="permissive")

# Read-only - blocks all write operations
kernel = KernelSpace(policy="read_only")

Custom Policies

Define your own rules:

from agent_os.policies import Policy, Pattern

policy = Policy(
    name="my-policy",
    blocked_actions=[
        "file_write",
        "file_delete",
        "database_drop",
        "shell_execute"
    ],
    blocked_patterns=[
        Pattern(r"\bpassword\b", "PII: password detected"),
        Pattern(r"\bssn\b", "PII: SSN detected"),
        Pattern(r"DROP\s+TABLE", "Destructive SQL blocked")
    ],
    allowed_domains=[
        "api.openai.com",
        "internal.company.com"
    ]
)

kernel = KernelSpace(policy=policy)

Or use YAML configuration:

# policy.yaml
name: my-policy
version: "1.0"

blocked_actions:
  - file_write
  - file_delete
  - database_drop

blocked_patterns:
  - pattern: "\\bpassword\\b"
    reason: "PII: password detected"
  - pattern: "DROP\\s+TABLE"
    reason: "Destructive SQL blocked"

allowed_domains:
  - api.openai.com
kernel = KernelSpace(policy="policy.yaml")

Agent Signals

Control agent execution with POSIX-style signals:

from agent_os import KernelSpace, SignalDispatcher, AgentSignal

kernel = KernelSpace(policy="strict")
dispatcher = SignalDispatcher()

# Register an agent
@kernel.register
async def long_running_agent(task):
    # ... long-running work ...
    pass

# Start the agent
agent_id = await kernel.spawn(long_running_agent, "big task")

# Pause the agent for human review
dispatcher.signal(agent_id, AgentSignal.SIGSTOP)

# Resume execution
dispatcher.signal(agent_id, AgentSignal.SIGCONT)

# Terminate immediately (non-catchable)
dispatcher.signal(agent_id, AgentSignal.SIGKILL)
Signal Description Catchable?
SIGKILL Terminate immediately No
SIGSTOP Pause for human review No
SIGCONT Resume execution N/A
SIGTERM Graceful termination Yes

Framework Integrations

Agent OS wraps existing agent frameworks with minimal changes:

LangChain

from langchain.agents import AgentExecutor
from agent_os.integrations import LangChainKernel

# Your existing LangChain agent
agent_executor = AgentExecutor(agent=agent, tools=tools)

# Wrap with Agent OS
kernel = LangChainKernel(policy="strict")
governed_agent = kernel.wrap(agent_executor)

# Use as normal - now with kernel enforcement
result = governed_agent.invoke({"input": "analyze this data"})

CrewAI

from crewai import Crew
from agent_os.integrations import CrewAIKernel

# Your existing crew
crew = Crew(agents=[agent1, agent2], tasks=[task1, task2])

# Wrap with Agent OS
kernel = CrewAIKernel(policy="strict")
governed_crew = kernel.wrap(crew)

# Run with governance
result = governed_crew.kickoff()

OpenAI Assistants

from openai import OpenAI
from agent_os.integrations import OpenAIKernel

client = OpenAI()
assistant = client.beta.assistants.create(
    name="My Assistant",
    model="gpt-4"
)

# Wrap with Agent OS
kernel = OpenAIKernel(policy="strict")
governed = kernel.wrap_assistant(assistant, client)

# Now governed
response = governed.chat("What's the weather?")
